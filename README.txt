MOVIE TRAILER WEBSITE 

This Project is part of "Udacity Python Foundation Nanodegree" Course which shows a web page listing movies and shows it's trailer. 

Getting Started:
These instructions will get you a copy of the project up and running on your local machine for development and testing purposed. 

Prerequisites:
* Any System running Windows, Linux, macOS
* Python 2.7.x or 3.6.x

Installing Python:
* Go to "https://www.python.org/downloads/" 
* Download and Install suitable version based on your OS.

Testing Project:
* Download the Zip File named "movie"
* Unzip it.
* Open the Command Interface on your system- e.g. Command Prompt on Windows or Terminal on Linux
* Navigate to the project directory and run the command "python entertainment_center.py"
